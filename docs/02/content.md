### 자료형 - 숫자

https://wikidocs.net/12

* 정수(int) : 정수
* 실수(float) : 소수점이 포함된 숫자, e 표기법도 가능. 다른 언어에서 실수를 float과 double로 나누는데, Python은 float 타입만 있음
* 8진수(int) : 정수, 0o로 시작하는 8진법 표현
* 16진수(int) : 정수, 0x로 시작하는 16진법 표현

```
a = 3
type(a)
```
```
b = 3.1
type(b)
```
